1. Protocol to automatically set Layout for NIS 6
The folder "NIS_install" has to be installed in "Nikon Service" (be careful on the spelling)

in NIS go to Macro > Run Macro From Files... and open the file C:\Nikon Service\NIS_install\Layout configuration.mac

2. If the customer is using GA3 (General Analysis 3) please check the "GA3_functions" folder and it Readme file.
